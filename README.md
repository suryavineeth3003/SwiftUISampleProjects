# 🌟 iOS SwiftUI Practice Projects

A collection of real-world **SwiftUI-based projects** built as part of hands-on learning and portfolio development. Each project is crafted with clean architecture (MVVM), modern SwiftUI features, and Swift best practices. Suitable for both interview preparation and real-world app development.

---

## 📱 Projects Included

### 1. ✅ Tip Calculator
A simple tip calculator where users can enter bill amount, select tip percentage, and see the total amount.

### 2. ✅ To-Do List App
Add, mark, and delete tasks with state management. Uses MVVM pattern and SwiftUI List.

### 3. ✅ Weather App 🌦️
Fetches real-time weather from a public API using `URLSession` and displays it beautifully with temperature, condition, and icons.

### 4. ✅ Movie Browser 🎬
Searches and lists movies from the OMDB API. Demonstrates working with JSON decoding, error handling, and dynamic lists.

### 5. ✅ Notes App 📝
Create, edit, and delete notes using Core Data as a persistence layer. Follows MVVM + SwiftUI integration with Core Data.

### 6. ✅ Finance Tracker 💰
Track income and expenses. Includes balance summary, transaction history, and persistent storage via `UserDefaults` or CoreData.

### 7. ✅ Habit Tracker
Track your daily habits. Uses Core Data to persist completion status, view history, and toggle habits.

### 8. ✅ Quiz App
Multiple-choice quiz with timer, dynamic score tracking, and result screen. Supports question randomization and restarts.

### 9. ✅ Image Grid View
Displays images in a responsive grid using `LazyVGrid` and `AsyncImage`. Supports adaptive layout and remote image loading.

## 💡 Technologies Used

- `SwiftUI`
- `MVVM Architecture`
- `CoreData`
- `URLSession & Combine`
- `AsyncImage`, `LazyVGrid`, `LazyHGrid`
- `UserDefaults` for lightweight persistence

- Networking with `URLSession` and `Codable`

- Handling `AsyncImage`, progress, and fallback UI
